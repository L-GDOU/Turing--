# 图灵CV2轮考核

## 1 使用OpenCV几何图形绘制
### 1.1 创建画布
创建 800×800 白色画布。
### 1.2 使用OpenCV几何图形绘制
绘制矩形，圆形，椭圆，三角形，五边形等多种几何图形。 
分开绘制几何图形，方便检测.

## 2 边缘检测与计数
### 2.1 对绘制的图形进行边缘检测
先对图像进行预处理（转化为灰度图，去除噪声），在进行Canny边缘检测得到二值化图形。
### 2.2 使用轮廓检测统计图形数量
对边缘检测结果进行闭运算，闭合图像开口再检测轮廓.
### 2.3在原图上标注检测到的轮廓
绘制绿色轮廓
